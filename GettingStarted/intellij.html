Including a jar file in an IntelliJ project


Let us assume that our Java project is called AmidstEmptyProject. In this example, our project only contains a single class called Main. First of all, click on the root folder of the project in the Project Window:



Create a new folder for saving all the jar files. For that, in the menu bar, go to File>New>Directory as shown in the following image:



A dialog will appear for entering the name of the new folder. For example, write lib and press the button OK.



A new empty folder will now appear in the project folder:




Now drag and drop the file AmidstToolbox-basic.jar from your file manager into the the lib folder:



Now we should include the libraries folder into the classpath of the project. For that, go to File>Project Structure in the menu bar:




A new window will appear with the Project Settings. In the left panel, select Libraries and click to the button + to add a new java library as shown in the following image:



A new dialog for selecting the folder will appear. Specify the path to the lib folder (it might be different to the one shown in the image below) and press OK.



Now select the modules in your project that will use the AMIDST Toolbox and press OK:




Finally, press OK in the Project Setting window and you will be able to use the functionality offered by AMIDST in your code project.
